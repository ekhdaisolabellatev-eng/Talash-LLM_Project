# Quick Start

1. Start backend:

```bash
cd milestone_3
pip install -r requirements.txt
python app.py
```

2. Start React frontend:

```bash
cd milestone_3/frontend
npm install
npm start
```

3. Open:

```text
http://localhost:3000
```

4. Login:

```text
admin / admin123
```

Use `user / user123` if you want a clean first-time workspace with no candidates.

5. Demo flow:

- Open Dashboard
- Review top ranked candidates
- Open Candidate Ledger
- Open a candidate detail report
- Queue a missing-information email
- Open Reports
- Mark candidate response as received
- Check Settings for rubric coverage
